Guía de Inicio Rápido de Geomajas¶

Geomajas es en sí mismo es un marco de aplicación que le permite construir soluciones GIS web muy diversas y avanzadas.

En este Live DVD de OSGeo hemos incluido una aplicación de Geomajas lista para usarse. Esta aplicación presenta un mapa, controladores de navegación, varias capas de datos, una barra de herramientas de cinta con herramientas SIG y el Árbol de Capas.

Para empezar¶

Geomajas está disponible como una demo web o como una aplicación independiente.

Para continuar usando la demo web visite http://www.geomajas.org/demo.
Para continuar usando la aplicación independiente, visita http://files.geomajas.org/downloads/2.3.0/geomajas-standalone-gwt2.zip.

Cuando se utiliza la aplicación independiente, ejecute cualquier startup.bat en Windows, o startup.sh en Linux y Mac. Entonces podrá acceder a la aplicación usando la siguiente URL: http://localhost:8080/geomajas/. Cuando haya terminado de usar la aplicación, puede finalizarla ejecutando shutdown.bat o shutdown.sh.

Uso de aplicaciones de Geomajas¶

Mirando a su alrededor¶

Puede arrastrar el mapa con el ratón para que el área deseada esté centrada en su pantalla.

Para obtener una mirada más cercana, puede usar la rueda central del ratón para ampliar o usar los botones de zoom que están en la esquina superior izquierda de la aplicación. También puede seleccionar un área para mostrar haciendo click en el ícono de la lupa y seleccionar el área deseada (por medio de selección rectangular).

Leyenda¶

En la parte superior derecha, se proporciona un cuadro de leyenda conmutable. Esta leyenda le da una vista general de todas las capas actualmente cargados en la aplicación, con la opción encenderlas o apagarlas. Esto también muestra información más detallada sobre cada capas con respecto a los iconos, coloreado…

Recopilación de información¶

Las aplicaciones de Geomajas tienen la capacidad para mostrar una variedad de información dentro de las capas. Haciendo click sobre el mapa se mostrará la información relacionada si está disponible. Haciendo click en el icono de información en la parte superior izquierda, se puede alternar entre información básica o detallada.

Personalizar aplicaciones Geomajas¶

Si desea utilizar esta aplicación de muestra para construir y personalizar su propia solución Web de SIG … visite http://www.geomajas.org/client-gwt. La página de descarga siempre cuenta con las mejores y más recientes aplicaciones de muestra, documentación, consejos y trucos para que usted mismo lo haga.
