Proj.4¶

Proj4 is a library that provides methods to transform between different coordinate reference systems. A geographic coordinate reference system allows all points on the earth to be described as a set of coordinates (such as latitude, longitude and elevation). Different systems are used to represent the 3 dimensional earth on a flat, 2 dimensional map.

Proj4 se usa en MapServer, GRASS GIS, PostGIS, Mapnik, Thuban, OGDI, TopoCad, OGRCoordinateTransformation y varios otros.

Proj4 ha sido traducido a partir de C a otros lenguajes <http://trac.osgeo.org/metacrs/wiki>. Estos subproyectos incluyen: Estas herramientas incluyen: * Proj4js (JavaScript) - usada en OpenLayers. * CS-Map (C++) - usada en MapGuide. * GeoTIFF/libgeotiff (C++). * Proj4J (Java). * SpatialReference.org - sitio web para recuperar las definiciones del sistema de coordenadas (obsoleto).

Características Principales¶

  • Proporciona el punto de transformación de un sistema de referencia de coordenadas a otro.
  • Incluye transformación entre datums (datos de referencia).
  • Gran cantidad de clases de proyección compatibles.

Estándares Implementados¶

Proj4 utiliza las definiciones del sistema de coordenadas dadas por el European Petroleum Survey Group (EPSG) < http://www.epsg.org/ >”. Bien conocido formato de texto (WKT).
