Guide de démarrage avec QGIS¶

QGIS est un logiciel SIG bureautique convivial qui vous permet de visualiser, gérer, éditer, analyser les données et composer des cartes imprimables.

Modifier le projet QGIS¶

Nous allons commencer par ouvrir un projet existant de QGIS et afficher/cacher des couches.

  1. Lancez QGIS de Geospatial ‣ Desktop GIS ‣ QGIS et sélectionnez Project ‣ Open dans la barre de menus.

  2. Choisir:file:QGIS-NaturalEarth-Example.qgs, puis appuyez sur Open

    • Vous devriez voir une carte du Monde.

  3. Cochez la case ne_10m_populated_places dans l’arborescence de couches.

    • Les villes s’affichent désormais en autant de points verts :

  4. Essayez de faire glisser les couches verticalement dans la légende et voyez comment cela affecte la visibilité des couches inférieures.

  5. Jetez un coup d’œil sur les outils de la barre d’outils. Essayez de déplacer la vue panoramique, les zoom avant et zoom arrière sur toute l’étendue. Trouver ces outils à côté de l’icone hand. Si les barres d’outils semblent encombrés, vous pouvez les faire glisser autour et les faire disparaître/apparaître par un clic droit. Vous pouvez également zoomer et dézoomer avec la molette de la souris et déplacer la vue panoramique avec un cliqué-glissé gauche.

Styliser une couche¶

Maintenant nous allons essayer de personnaliser le style de la carte.

  1. Zoomez un peu sur la carte, puis double cliquez sur ne_10m_rivers_lake_centerlines dans l’arborescence de couches.

#. In the Layer Properties dialog on the Style tab click on the Color to select a different color, say yellow.

  1. Pressez OK.

    • Notez que les rivières sont maintenant affichées dans votre nouvelle couleur.

Créez un nouveau projet QGIS¶

Nous allons maintenant créer un nouveau projet QGIS et charger nos propres données.

  1. Choisir Project ‣ New. On vous demandera s’il faut enregistrer le projet précédent, vous pouvez appuyer sur Close without Saving – Fermer sans enregistrer les modifications.

#. Click Layer ‣ Add Layer‣ Add Vector Layer… Or click on the “Add Vector Layer” button (the one with the “V” shape, in the red rectangle in the image). You can also use the keyboard shortcut “Ctrl+Shift+V”

  1. Accédez au dataset :file:”/home/user/data/natural_earth2/ne_10m_admin_0_countries.shp”.

  2. Presse Open puis Open à nouveau.

    • Vous devriez voir tous les pays du monde.

Se connecter à une base de données spatiale PostGIS¶

Nous allons maintenant inclure une couche d’une base de données Postgres.

  1. Dans la liste des couches sur la gauche, décochez la case de visibilité ne_10m_admin_0_countries pour la masquer temporairement.
  2. Choisissez Layer ‣ Add Layers ‣ Add PostGIS Layers….
    • Vous pouvez également cliquer sur l’icône avec la tête de l’éléphant dans le panneau de gauche ou utilisez le raccourci clavier “Ctrl+Shift+D”
    • Five Postgis databases are already available; we will be using the Natural Earth database. If you wanted to connect to a different database, you would select the New button and fill in the database parameters.

#. Select the « Natural Earth » connection and press Connect. Then click on the Public schema to deploy it:

  • La liste des tables de la base de données s’affiche.

  1. Select ne_10m_populated_places and click Add.

    • Pour plus de détails sur l’utilisation des bases de données PostGIS, voir le PostGIS Quickstart.

  2. Zoomez sur les États-Unis en utilisant la roulette de la souris et un cliqué-glissé gauche de souris pour naviguer.

  3. Faites un clic droit sur ne_10m_populated_places dans la liste des couches pour obtenir un menu contextuel, puis sélectionnez Properties.

  4. Let’s represent one of the database attributes in the data as a bubble plot. In the middle of the Style tab, drag the Transparency slider to 50%. Click on the small button at the right of the size field and hover over the option within Attribute field, then choose scalerank (it’s near to the beginning of the list). Then click Ok.

  5. Vous pouvez alors cliquer sur le bouton Requête dans la barre d’outils (curseur flèche avec un « i » bleu) et ensuite sur les bulles de la carte pour afficher des informations sur les différentes villes.

Utilisation de la boîte à outils GRASS¶

Il y a eu de nombreux plugins écrits pour QGIS qui étendent les fonctionnalités principales de QGIS. Un des plus puissants est le plugin GRASS, qui s’approvisionne dans les centaines de modules de traitement géospatial disponible à partir de GRASS GIS.

Note

Vous devrez modifier les autorisations du dossier GRASS data afin d’effectuer les étapes suivantes. S’il vous plaît, ouvrez un terminal et exécutez la commande suivante : « sudo chmod -R 775 /usr/local/share/grass/nc_basic_spm_grass7 »

  1. Effacer l’ardoise avec Project ‣ New.

  2. Choisissez Plugins ‣ Manage and Install Plugins…, puis faites défiler vers le bas ou ou saisissez grass dans la zone de recherche, puis sélectionnez le plugin GRASS 7

    • Notez qu’une nouvelle icône GRASS a été ajoutée à la barre d’outils, une fenêtre ancrée nommée « GRASS Tools » est apparue sur la droite de la zone réactive et un nouvel élément de menu GRASS a été ajouté au menu Plugins .

  3. Connectez vous à un espace de travail existant de GRASS avec Plugins ‣ GRASS ‣ Open mapset.

    • La base de données SIG GRASS (Gisdbase) a déjà été définie à /home/user/grassdata sur le disque pour vous.

  4. Au sein de la base de données centrale de GRASS se trouvent un certain nombre d’ensembles de données échantillon. Nous allons charger l’emplacement de la Caroline du Nord et le jeu de cartes user1 qu’il contient. Choisissez l’emplacement de nc_basic_spm_grass7 et le jeu de cartes de travail user1, puis cliquez sur Ok.

  5. Pour ajouter une carte raster à la liste des couches QGIS, placez-vous depuis le panneau d’exploration de QGIS dans le dossier Home/grassdata/nc_basic_spm_grass7.

    • Dans le jeu de cartes PERMANENT, sélectionnez la carte elevation et double-cliquez dessus pour l’ajouter à la carte.

  6. Ajoutez une autre couche raster GRASS, cette fois la carte geology du jeu de données PERMANENT.

    • Double-cliquez sur la carte geology dans la liste de couches de QGIS et dans l’onglet Transparence régler sa transparence globale à 70 %.

  7. Pour ajouter une carte vectorielle, sélectionnez une couche vectorielle dans le navigateur de QGIS, similaire aux étapes précédentes.

    • Dans le jeu de cartes PERMANENT, sélectionnez la carte de roadsmajor avec un double-clic.

  8. Change the layer order if necessary (roadsmajor, geology, elevation).

Le plugin vous donne également accès à de nombreux puissants modules d’analyse et outils de visualisation GRASS :

  1. Depuis le menu principal sélectionnez Plugins ‣ GRASS ‣ Open GRASS tools et faites glisser le bord pour rendre la fenêtre un peu plus grande.

    • Une longue liste d’outils d’analyse apparaît. Accédez à l’ onglet Modules Tree et sélectionnez Region settings ‣ g.region.multiple.raster. Cliquez dessus, un nouvel onglet va s’ouvrir. Selectionnez elevation dans la liste du menu et pressez Run. La carte elevation a désormais une ligne rouge autour d’elle, l’étendue des limites de la computational region de GRASS

  2. En retournant dans l’onglet Modules Tree de la fenêtre GRASS Tools , aller vers le bas Raster`et sélectionnez :menuselection:`Surface Management ‣ Generate Vector contour lines.

  3. Dans le nouvel onglet module qui s’ouvre automatiquement, sélectionnez la carte elevation comme entrée.

  4. Ajouter quelques courbes de niveaux (par exemple, 20, 40, 60, 80, 100)

  5. Sélectionnez le nom de couche de sortie (par exemple contour_lines), puis cliquez sur Run.

Utilisation de la boîte à outils Géotraitements¶

A core plugin for QGIS which opens the door to a large family of processing tools is the Processing Toolbox (formerly named the SEXTANTE Toolbox). It acts as a standardized wrapper around a number of other sets of tools.

  1. Choisissez Processing ‣ Toolbox.

    • Une nouvelle barre d’outils s’ouvre sur le côté droit de l’écran avec de nombreux outils de traitement au choix. Prenez le temps d’y jeter un oeil.
    • Vous devrez peut-être activer un fournisseur de traitement afin de pouvoir l’utiliser. La capture d’écran suivante montre comment activer le support des outils de traitement GRASS GIS 7. Importation de données OpenStreetMap¶

  1. Ouvrez l’émulateur de terminal LX du principal Accessories menu.

    • Coupez et collez les commandes suivantes dans la fenêtre de Terminal pour créer une copie de travail des données OSM dans le répertoire home :

      cp data/osm/feature_city.osm.bz2 .
bzip2 -d feature_city.osm.bz2

  2. Dans QGIS, choisissez Project ‣ New. Si vous aviez la boîte à outils de traitement ouverte, vous pouvez la fermer.

  3. Choisissez Vector ‣ OpenStreetMap ‣ Import topology from XML.

  4. Cliquez sur le bouton « … » à côté « … » et sélectionnez le fichier feature_city.osm que vous venez de copier dans le répertoire de base. Le nom « Output SpatialLite DB file » sera automatiquement défini. Cliquez sur Ok pour convertir le groupe de données au format de SpatiaLite et créer une connexion à la DB SpatialLite dans QGIS.

  5. Ensuite, il faut extraire des points, lignes et zones, puis ajoutez la topologie à chacune de ces trois nouvelles couches. Pour ce faire, nous avons besoin d’exécuter l’outil trois fois. Sélectionnez Vector ‣ OpenStreetMap ‣ Export toplogy to SpatiaLite`et utilisez le bouton « … » pour sélectionner le fichier nouvellement créé `feature_city.osm.db L” Output layer name sera automatiquement rempli pour vous selon le Export type sélectionné. Cliquez sur le bouton Load from DB pour charger les balises disponibles. Pour la couche de « points », cochez la case de amentity; pour la couche de « polylines » cochez la couche highway ; et pour la couche de « polygon » sélectionnez la couche building. Vous pouvez changer l”Output layer name afin de refléter les balises de fonction de ce que vous avez sélectionné. Lorsque vous êtes prêt, appuyez sur Ok pour charger la couche. Vous devrez appuyer à nouveau sur le Load from DB après avoir changé le type d’exportation de points pour les polylignes et polylignes aux polygones.

  6. Une fois que la topologie est chargée, vous pouvez également affiner la couche SpatiaLite en interrogeant seulement certaines fonctions. Sélectionnez Layer ‣ Add Layer ‣ Add SpatiaLite Layer… dans le menu de la liste Databases et sélectionnez feature_city@… puis cliquez sur Connect. Double-cliquez sur la table feature_city_polylines et puis double-cliquez sur « highway » pour commencer à construire votre requête SQL. Cliquez ensuite sur le bouton, = puis le bouton All et double-cliquez sur motorway de la liste de valeurs. Cliquez sur le bouton Test pour vérifier le résultat et enfin cliquez sur Ok. Dans la fenêtre Add SpatiaLite Table cliquez sur Add pour limiter le rendu aux autoroutes majeures. Vous pouvez répéter cette opération avec les nouvelles couches pour afficher les types de routes différents avec des largeurs et des styles différents.

  7. You can now explore this rich dataset. Use the i information cursor button in the QGIS toolbar to query individual map features.

Choses à essayer¶

  • Essayez d’afficher les sources de données avec le QGIS Data Browser dans le menu Geospatial ‣ Databases
  • Essayez de publier votre carte QGIS sur le web en utilisant QGIS Map Server dans le menu Geospatial ‣ Web Services.
