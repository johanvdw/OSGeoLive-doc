Guide de démarrage de ZOO Project¶

Le ZOO-projet fournit un cadre de développement facilité pour pour la création et le chaînage de géotraitements par le Web (WPS - Web Processing Service). Un WPS fournit l’accès web aux fonctions qui exécutent des algorithmes spatiaux.

Ce guide de démarrage rapide décrit comment:

  • Appliquer le processus à un WFS depuis un navigateur web.

Lancement¶

  • Pour exécuter la démo ZOO Project à partir du DVD OSGeoLive, cliquez sur le lien de « ZOO Project », vous le trouverez dans les éléments de menu sous « Geospatial » > « Web Services ».
  • Firefox ouvrira la page de lancement de démonstrations de ZOO Project. De là, vous pouvez décider quelle démonstration que vous souhaitez utiliser.

Outils spatiaux OGR¶

  • Cliquez sur la carte pour sélectionner une fonction de la GeoServer serveur WFS, la fonction s’affiche en bleu, puis cliquez sur processus de géométrie simple comme Buffer, Centroid, ConvexHull oo Boundary et le résultat apparaîtra en jaune.
  • Maintenant, sélectionnez une autre entité près de la première que vous avez sélectionné, puis cliquez sur plusieurs boutons de processus de géométries: Union, Intersection, différence symétrique ou Différence pour obtenir le résultat affiché en rouge sur la carte.

Triangulations CGAL¶

  • De l’application de triangulations CGAL, il suffit d’appuyer sur le bouton Voronoï ou Delaunay pour afficher sur votre carte, le résultat de l’exécution du service WPS, comme illustré ci-dessous.

Applications OTB¶

  • From the OTB application demonstration interface, you have to fill the form on the left to configure the WPS service execution. Once, you have selected the value for your processing, you can then press the Execute button at the bottom of the left form. In case you choose image/png as the output format, you will see the result of your processing displayed on the map. You can see below a sample KMeansClassification execution when selecting the image/png output format.

Ensuite ?¶

  • Vous pouvez accéder à des capacités du serveur ZOO WPS :
    http://localhost/cgi-bin/zoo_loader.cgi?Request=GetCapabilities&Service=WPS
  • Pour toute autre question, vous pouvez contacter la liste de diffusion ZOO Project : zoo-discuss@lists.osgeo.org
