Inicio rápido del Servidor QGIS¶

El Servidor de QGIS ofrece un servicio de mapas web (WMS) basada en las bibliotecas de la aplicación de escritorio QGIS.

Esta Guía de Inicio Rápido describe cómo:

  • actualizar un proyecto existente de QGIS y renderizar con el Servidor de QGIS
  • Crear un nuevo proyecto QGIS y abrir el WMS directamente en un navegador
Documentación:
  • https://docs.qgis.org/2.14/en/docs/user_manual/working_with_ogc/ogc_server_support.html

Guia para la instalación del Servidor en Debian/Ubuntu:

  • apt-get install qgis-mapserver libapache2-mod-fcgid
  • -> http://localhost/cgi-bin/qgis_mapserv.fcgi
Cliente de Web de QGIS:
  • https://github.com/qgis/qgis-web-client

Inicie el visualizador de GeoExt¶

OSGeoLive contiene un ejemplo de visor web basado en el framework de GeoExt Javascript y utilizando servidor de QGIS para representar el mapa.

  • Haga click en Geospatial ‣ Web Services ‣ QGIS Server

Deberá ver una ventana del navegador con un visor web que muestra un mapa del mundo.

Alternativamente, se puede acceder a la aplicación mediante un navegador a través de http://localhost/qgis_server

Ahora vamos a cambiar el estilo del mapa utilizando QGIS.

Editar el proyecto en QGIS¶

  1. Haga clic en: menuselection: “Geoespacial–> Desktop GIS–> QGIS”.
  2. Seleccione Project ‣ Open… from the menu bar.
  3. Elegir QGIS-NaturalEarth-Example.qgs y presione Abrir.
    • Debería ver el mismo mapamundi de QGIS. Ahora cambiaremos de estilo del río y veremos el resultado en el visualizador del mapa web.
  4. Haga doble clic en ne_10m_rivers_lake_centerlines en el árbol de capas.
  5. Cambiar el color en Opciones de Esquema para un tono de azul diferente.
  6. Presione OK.
    • Los ríos se renderizan ahora en su nuevo color.
  7. Guarde el proyecto con Project ‣ Save.
  8. Abrir el navegador web
  9. Hacer Acercamiento
    • Ahora deberá ver los ríos en el mismo nuevo color.

Cree un nuevo proyecto QGIS¶

  1. Abrir de nuevo QGIS
  2. Elegir Project ‣ Nuevo.
  3. Hacer click en Capa ‣ Añadir capa….
  4. Navegue al conjunto de datos /home/user/data/natural_earth2/ne_10m_admin_0_countries.shp.
  5. Presione Abrir.
    • Usted debe ver todos los países de mundo.
  6. Guarde el proyecto con Project ‣ Save.
  7. Utilizar nombre de archivo /home/user/world.qgs y prensa Save.
  8. Abrir el WMS URL.
    • Usted debe ver todos los países de mundo, esta vez por Servidor de QGIS.

¿Qué sigue?¶
