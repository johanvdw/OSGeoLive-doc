Deegree Quickstart¶

deegree ist eine Java-Applikation für die Bereitstellung von Karten (und Vektordaten), damit diese von anderen Clients dargestellt werden können.

Diese Kurzanleitung beschreibt:

  • Starten von deegree Webservices
  • Konfigurieren von deegree Webservices über die deegreee Services Console
  • Verwendung des deegree-inspire-workspace als Ausgangspunkt

Projektbeschreibung¶

deegree ist Open-Source-Software für Geodateninfrastrukturen und das Geospatial Web.

Verfügbare Komponenten sind:

  • Web-Services: WFS, WMS, WMTS, CSW, WPS

Diese Kurzanleitung beschreibt:

  • Starten der deegree 3 Webservices auf der Live DVD
  • Verwendung der deegree 3 Services Console, um die Beispielkonfigurationen zu aktivieren
  • Mehr Informationen

Über deegree 3¶

deegree 3 ist die neue Generation des deegree Java-Frameworks für GIS-Anwendungen und OGC-Service-Implementierungen. deegree kann verwendet werden, um web-basierte Geodateninfrastrukturen (GDI) zu bauen.

Starten von deegree 3 Webservices¶

Wähle ‚Start deegree‘ aus der Geospatial ‣ Web Services ‣ deegree menu. Die Anwendung ist nach kurzer Zeit gestartet.

deegree 3 Services Console¶

Beim Starten von * Deegree * zum ersten Mal Firefox sollte geöffnet werden (ansonsten manuell öffnen und zeigen Sie Ihren Browser auf Http://localhost:8033). Bitte warten Sie, bis du siehst die * Deegree 3 Service-Konsole *. Dies ist eine einfache Web-basierte GUI für die Bearbeitung der Konfigurationsdateien der sogenannten * Deegree Arbeitsbereich *.

Deegree 3 Arbeitsbereiche¶

deegree 3 uses configurations conformant to all deegree 3 based webservices. Those configurations are stored in the so called workspace.

Im Inneren der * Deegree-Dienste-Konsole * finden Sie einen Abschnitt, der im Preis inbegriffen und Ready-to-Use Arbeitsbereiche verwalten. Um diesen Abschnitt zu gelangen, klicken Sie auf * „Arbeitsbereiche“ * auf der linken Seite.

Du bist jetzt in der Workspace-Management der laufenden * Deegree * Instanz. Sie können sehen, welcher Arbeitsbereich gerade aktiv ist (nur eine zu einem Zeitpunkt aktiv sein kann) Arbeitsbereich, inaktive lokal verfügbare Arbeitsbereiche und offizielle Arbeitsbereiche, die importiert werden können, wenn Sie eine Internetverbindung haben.

Bitte starten Sie die * Deegree-inspirieren-Arbeitsbereich * jetzt.

Deegree 3 Arbeitsbereiche auf OSGeoLive GIS Disc¶

  • deegree-workspace-inspire

In diesem Workspace berfindet sich eine Beispielkonfiguration für INSPIRE View- und Download-Services. Er besteht aus einem transaktionalen WFS (2.0.0 und 1.1.0) für alle Annex I Datenthemen und einem WMS (1.3.0 und 1.1.1), der alle drei Layer der Annex I Datenthemen anbietet. Enthalten sind einige harmonisierte niederländische administrative units, cadastral parcels und addresses. Der WFS fungiert in diesem Beispiel als ein INSPIRE Download-Service (direct access) und gibt valides, harmonisiertes INSPIRE GML zurück. Zusätzlich unterstützt er mehrere Request Typen und Queries.

Tipp

Dieser Workspace lädt harmonisiertes INSPIRE GML aus lokalen Dateien. Das Backend kann leicht zu PostGIS, Oracle Spatial oder Microsoft SQL Server geändert werden.

Nach der Aktivierung der * Deegree-Arbeitsbereich-inspirieren * Sie können einen einfache Karte-Client öffnen, indem Sie auf * ‚finden Sie unter Ebenen‘ *. Der Dienst ist so konfiguriert, dass um die Daten und eine Grundkarte (dargestellt durch ‚OpenStreetMap < http://openstreetmap.org/ >‘ _) zu visualisieren.

Ein Klick auf die ** + ** auf der rechten Seite öffnet eine Folienbaum zeigt alle verfügbare Ebenen. Sie können durch die Deegree-WMS hier serviert INSPIRE-Layer aktivieren.

Tipp

Der Karten-Client basiert auf OpenLayers. Durch das Halten der linken Maustaste und Ziehen der Karte kann über die Karte navigiert werden. Zoomen erfolgt via Mausrad oder über die Navigationselemente auf der linken Seite.

More information and step-by-step tutorials about the INSPIRE workspace can be found in the deegree documentation.
