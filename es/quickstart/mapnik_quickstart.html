Guía de Inicio Rapido de Mapnik¶

Mapnik es un conjunto de herramientas para desarrollar aplicaciones de mapeo. Sobre todo, Mapnik trata sobre la elaboración de mapas hermosos. Es fácilmente extensible y adecuado para el desarrollo web y en escritorio.

Mapnik & Python¶

Mapnik y sus enlaces de python están instalados y listos para ser utilizados para scripting en esta máquina.

Crear mapas en python es fácil con Mapnik. Teclee python en la línea de comando para entrar en un intérprete de python e intente esto:

>>> import mapnik, os
>>> m = mapnik.Map(600,400)
>>> style = '/usr/local/share/mapnik/demo/population.xml'
>>> mapnik.load_map(m,style)
>>> m.zoom_all()
>>> mapnik.render_to_file(m,'map.png')
>>> os.system('xdg-open map.png')

Nota

El código anterior depende de tener una hoja de estilo XML que Mapnik puede leer , definiendo las capas desde donde se leen los datos y los estilos para aplicar a esas capas. Usted puede crear uno de éstos dentro de QGIS con el complemento de Quantumnik: http://plugins.qgis.org/plugins/quantumnik/

Mapnik & Leaflet¶

Además una aplicación demo básica está disponible, que pone de relieve utilizando Mapnik servir azulejos en un mapa de la web de OpenLayers en el esquema de azulejo OSM/Google.

La demostración utiliza una servidor de teselas diseñado para Mapnik llamado «TileStache»

  • Más información sobre TileStache: http://tilestache.org/

Para ejecutar la demo sólo tienes que seguir estos pasos:

  1. Click Desktop ‣ Spatial Tools ‣ Start Mapnik & TileStache
  • El servidor debe comenzar en una ventana de terminal (que permanece abierta y muestra información básica de depuración).
  1. Asegúrese de que el servidor trabaja solicitando una tesela desde el servidor en http://localhost:8012/example/0/0/0.png
  2. Luego visite la página demo de la aplicación de Leaflet:
  • Consultar la página en http://localhost/leaflet-demo.html
