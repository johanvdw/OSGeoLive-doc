Letöltés¶

The latest stable release is OSGeoLive 12.0 and can be downloaded from: https://sourceforge.net/projects/osgeo-live/files/12.0/

Válassza az önnek megfelelő image-t:

osgeolive iso: 4.0 GB

Egy Lubuntu alapú betölthető rendszer, melyet egy 8 gigás USB meghajtóra (gyorsabb és ezt javasolják) vagy DVD-re (olcsóbb) másolhatunk.

osgeolive-vm: 3.1 GB

Ez egy osgeolive alapú, előkészített virtuális gép (.vmclk), mely használható a VirtualBOX, VMWare, KVM és más virtuális gép alkalmazásokban. A 7-Zip segítségével tömörítettük.

amd64 vagy i386 architektúra

ISO imagek állnak rendelkezésre a legújabb amd64 hardverhez (javasolt) ugyanúgy mint a régebbi i386 hardverhez.

md5 ellenőrzőösszeg

Az md5 ellenőrzőösszeggel a letöltés hibátlanságát ellenőrizheti.

Rendszer követelmények

Minimális javasolt rendszer erőforrások: 1 GB RAM (2GB jobb a Java alapú alkalmazások kipróbálásához), 1GHz i386 vagy amd64 kompatibilis CPU. Nincs szükség merevlemezre. Mac felhasználóknak előnyös a 3 gombos USB egér.
