La fundación OSGeo¶

Empoderando a todos con el software libre geospacial

La Fundación para el Código Abierto Geoespacial (OSGeo), es una organización sin ánimo de lucro cuya misión es la promoción del desarrollo colaborativo de tecnologías geoespaciales abiertas siendo una organización inclusiva y dedicada a una filosofía abierta y participativa y al desarrollo dirigido mediante una comunidad participativa.

Las actividades de difusión de OSGeo incluyen:

  • Capítulos Locales: grupos de apoyo locales, regionales y por idiomas para promover actividades de comunidad.
  • GeoForAll: Más de 100 laboratorios educativos de todo el mundo que trabajan con otros socios para hacer la educación geoespacial y otras oportunidades accesibles para todos.
  • OSGeoLive: Una distribución de software geoespaciales de código abierto de distribución de geoespaciales establecidos, preconfigurado con datos de ejemplo y tutoriales, listo para probar.
  • FOSS4G: Conferencia anual e internacional para Software Libre y de Código Abierto Geoespacial, así como muchos programas regionales y locales events.
